# planet-management
Simple CRUD operations API

Technologies:
    java - 11
    spring - 2.6.1
    postgresql - 13.5
    hibernate - 5.6.1
    maven - 3.8.4

How to run application:
1) Change application.properties based on your postgre config
2) Run PlanetmanagementApplication or run tests
3) Search REST APIs folder for JSON endpoints
